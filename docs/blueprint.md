# **App Name**: Khatupati Flow

## Core Features:

- Production Lot Creation & Costing: Create new production lots ('Pieces'), define fabric components (Kurta, Salwar, Dupatta), rates, and dying charges. Automatically calculate total landing costs based on 'Surat Math' logic.
- Embroidery Jobwork Management: Track pieces sent and received for embroidery, assign vendors and rates, and attach primary design photos (via web file input/camera API). Automatically calculate quantity loss.
- Value Addition Jobwork Management: Manage the second stage of production (VA), assigning vendors and rates. Automatically pull quantities from completed embroidery stage. Attach secondary design photos.
- Real-time Inventory Tracking: View and manage the inventory of 'Pieces' across four distinct states: In-House (Raw), At Embroidery, At Value Addition, and Finished Stock.
- Vendor Stock Overview Dashboard: A central dashboard displaying current stock quantities held by each vendor for both Embroidery and Value Addition stages, facilitating quick oversight.
- Quick Receive Action: Streamlined one-tap option within job lists to mark received quantities, accelerating the production workflow.
- AI Design Insight Tool: Utilize an AI tool to analyze uploaded design photos and automatically suggest keywords, themes, or classifications, improving searchability and inventory categorization for design elements.

## Style Guidelines:

- Primary interactive color: A deep, professional indigo blue (#173880) for key interactive elements and branding, conveying reliability and organization.
- Background color: An ultra-light, subtle blue-gray (#F9FAFC) ensuring high readability and a clean workspace, adhering to the requested 'Light UI' and 'White/Light Gray backgrounds'.
- Accent color: A vibrant, contrasting violet-blue (#634CCB) to highlight critical information, notifications, or secondary calls to action, complementing the deep primary blue.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, techy, and bold feel, ensuring clarity for titles and prominent text.
- Body font: 'Inter' (sans-serif) for a neutral, machined, and objective look, optimizing readability for data, forms, and general text within the application.
- Employ a consistent set of clean, geometric, outlined icons to represent actions and navigation clearly, enhancing intuitive understanding and visibility in a warehouse environment.
- Mobile-first layout with high-contrast text to ensure optimal visibility, especially in varied lighting conditions. A prominent 'Home' or 'Dashboard' icon will be consistently placed in the top-right header of every screen for instant navigation, and dedicated search/filter bars will be available on relevant lists.
